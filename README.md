＜AIを使用した場面＞
・横の棒グラフのプログラム
"sometimes"という少し長めのラベルを使用したかったためloneliness_levelをy軸にしたく、横の棒グラフを表示させたかった。しかし授業では縦の棒グラフのみ取り扱っていたためAIを使用し横の棒グラフを表示させた。
・エラーの解決
最初、エクセルで開いたデータは表としてきれいに表示されていたが、CSVファイルとしてVScodeで開くと並びがバラバラになっており、headerを1から順に試していってもうまく表示されなかった。また、インデントがタブとスペースの混在でズレており表示されなかったり、エラーが出たりした。
・理解が曖昧な部分のコードの説明のために使用。AIで調べた部分はコメントを追加した。
