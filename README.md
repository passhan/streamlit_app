   グラフの色分けをON/OFFできるようにするために使用した。「性別ごとの差を比較したいとき」と全体の傾向だけを見たいとき」の両方に対応できるようにした。ONにすると性別ごとの色分けが有効になり、OFFにすると単色表示になる。
